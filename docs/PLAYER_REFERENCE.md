## 装備スキル（5種）

| スキル | 誰が | 一言 |
|--------|------|------|
| 偽位置 | 逃走者 | 約20秒、すべての暴露がおとり近くに出る |
| 偽情報暴露 | 鬼 | 本物そっくりの名前付き暴露を地図に置く |
| 体投げ | 鬼 | 人形の位置を基準に捕獲・暴露が動く |
| 捕獲結界 | 逃走者・鬼 | 範囲内の全員を拘束。効果中のみ離脱リスク |
| 鬼化・人化 | 人狼 | 姿を切り替えて陣営に応じた追跡・拘束 |

### 偽位置（逃走者）

- 進行方向の先（約38m）におとり。約20秒持続、再使用まで約72秒。
- **名前付き・匿名・定期暴露はすべておとり近傍**に出る。
- 鬼の体投げ稼働中は人形側が優先。

### 偽情報暴露（鬼）

- 自分 or ランダム逃走者を選び、プレイエリア内に名前付き暴露を設置。
- 相手からは偽とは分からない（理由タグなし）。匿名痕跡ではない。
- 設置成功後、再使用まで約75秒。

### 体投げ（鬼）

- 約90m以内に人形を設置。回収まで捕獲・暴露の基準は人形。
- 約12秒以内に約15m以内で回収しないと人形位置が名前付き暴露。
- 人形稼働中は**他スキル不可**。回収後、再使用まで約75秒。

### 捕獲結界

- 半径約55m・持続約24秒・再使用まで約80秒。設置射程約90m。
- 発動時に範囲内にいた**全員**（自分・味方・敵）の画面に結界が表示され、拘束される。
- **効果中**に円外へ出ると名前付き暴露（猶予約10秒後、捕獲可能なら脱落）。
- **終了後**は結界消滅・拘束解除。旧位置から離れても暴露なし。
- **接触拘束が時間切れで解けたあと**でも、結界を設置して拘束し直せる。結界の拘束中はごく至近（GPS約12m / BLE接触）で捕獲可能（接触圏の約4秒は不要）。
- 鬼陣営＋鬼化中の人狼の結界は捕獲不可（パニック・拘束のみ）。

### 鬼化・人化（人狼）

- **自動切替まで**と**切替可能まで**は別タイマー（HUDに別チップ表示）。
- **自動切替:** 前回の切替（自動・任意どちらでも）から `min(15分, 試合時間÷2)` で自動トグル（通知なし）。
- **切替可能まで:** 任意にボタンで切り替えられるまでの待ち＝自動切替間隔÷3。
- **自動切替の直後**は次に切り替えられるまでがやや長い（任意待ち + 任意待ち÷2）。
- 人陣営＋鬼化＝捕獲可。鬼陣営＋鬼化＝パニック・拘束のみ。告発不可。

## パニック（感染）

- 接触圏より外側の環に約6秒 → 約22秒のパニック状態。
- 約7秒ごとに**匿名痕跡**（名前付き暴露ではない）。脱落なし。
- 偽位置中は痕跡もおとり近傍。
- 文言例: 「鬼が近くにいます」「あと少しでパニックです」「このままだとパニックになります」。

---

## 接触拘束・捕獲（鬼の基本）

- 接触圏（約35〜95m）に約4秒 → **接触拘束**（拘束円約45〜110m・22〜75秒）。
- 拘束中: GPS約12m or BLE接触で即捕獲。拘束円外約10秒で暴露→捕獲。
- **拘束円は逃走者の画面のみ**（鬼にはピン位置として同期しない）。捕獲結界は範囲内全員の画面に円が出る。
- 拘束が時間切れで解けただけでは捕獲にならない。再び接触圏4秒、または捕獲結界で拘束が必要。

**スキル捕獲結界とは別物。**

---

## 鬼側の接近情報

- 逃走者の**常時 GPS ピン**は共有しない（プライバシー設計）。
- 逃走者端末が Firestore members に **`proximityBand`**（`far` / `near` / `contact` 等）を載せる。鬼は粗い接近シグナルとして参照できる。
- パニック・匿名痕跡・名前付き暴露は地図上の情報源。

## 情報の読み方

| 種類 | 意味 |
|------|------|
| 名前付き暴露 | 「○○がここにいた」。信頼度高いが今もそことは限らない |
| 匿名痕跡（?） | 「誰かがいた」。量で補う |
| 定期匿名 | 試合時間に連動した間隔で1人が匿名暴露 |

---

## 役職配分（オンライン）

- 役職はホストが `match_start` に書いた**1つのスナップショット**を全員が読む（端末ごとの再抽選なし）。
- **2人:** 必ず鬼1 + 逃走者1（人狼なし）。
