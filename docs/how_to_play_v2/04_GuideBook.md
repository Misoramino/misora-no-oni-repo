# 5. 勝ち方

## Section Header

タイトル:

```text
勝ち方
```

一言:

```text
人側は生き残るか本鬼を当てる。鬼側は人側を全員捕まえます。
```

## Diagram

```text
人陣営
├─ 時間切れまで生存
├─ 本鬼を告発
└─ 本鬼を全員脱落させる

鬼陣営
└─ 人側生存者を0人にする
```

## Card: 人陣営の勝ち

タイトル:

```text
人陣営の勝ち
```

一言:

```text
逃げ切る・当てる・本鬼を消す、のどれかで勝ちです。
```

本文:

```text
人陣営は、次のどれかを達成すると勝ちです。

・制限時間まで人側の生存者が1人以上残る
・告発施設で本鬼を当てる
・本鬼が全員脱落する

人狼が鬼陣営に残っていても、本鬼がいなければ人陣営の勝ちです。
```

## Card: 鬼陣営の勝ち

タイトル:

```text
鬼陣営の勝ち
```

一言:

```text
人側の生存者を0人にすれば勝ちです。
```

本文:

```text
鬼陣営は、逃走者を捕獲し、人側の生存者を0人にすると勝ちです。

告発に成功される前に、人側を追い詰めましょう。
```

## Card: 脱落と勝敗は別

タイトル:

```text
脱落しても終わりではありません
```

一言:

```text
個人の脱落と、陣営の勝敗は別です。
```

本文:

```text
捕獲・告発失敗・エリア外などで脱落しても、試合にはまだ関われます。

脱落後は、陣営に応じて第二ゲームに入り、味方を助けたり、相手を妨害したりできます。
```

## Detail: 試合中止

見出し:

```text
試合中止
```

本文:

```text
試合中止は、参加者の過半数投票またはホスト操作で行われます。

中止時は勝敗・戦績・トロフィーは付与されません。

軌跡保存に同意している場合、ギャラリーには「試合中止」として保存されることがあります。
```

```text
ライブ位置は見えません。痕跡と暴露から相手を読みます。
```

## Diagram

```text
ライブ位置 ×

📍 名前付き暴露
「○○がここにいた」

？ 匿名痕跡
「誰かがここにいた」
```

## Card: ライブ位置は基本見えない

タイトル:

```text
ライブ位置は基本見えません
```

一言:

```text
地図に出るのは、現在地ではなく手がかりです。
```

本文:

```text
ONI PINでは、仲間や敵のライブ位置が常に表示されるわけではありません。

地図に出るのは、監視カメラ、パニック、情報屋、エリア外などで発生した手がかりです。

「今ここにいる」と決めつけず、「少し前にここにいたかもしれない」と読んでください。
```

## Card: 名前付き暴露

タイトル:

```text
名前付き暴露
```

一言:

```text
「○○がここにいた」と分かる強い情報です。
```

本文:

```text
名前付き暴露は、プレイヤー名が付いた位置情報です。

エリア外に長くいる、告発に失敗する、情報屋に見つかる、偽情報暴露を受けるなどで発生します。

名前が出るため強い情報ですが、暴露されたあとに移動している可能性があります。
```

詳細:

```text
主な発生源:

・エリア外の初回暴露
・エリア外の再暴露
・告発失敗
・鬼の情報屋使用
・偽情報暴露
・体投げ失敗
・拘束円や結界からの脱出失敗
```

## Card: 匿名痕跡

タイトル:

```text
匿名痕跡
```

一言:

```text
「誰かがここにいた」とだけ分かる情報です。
```

本文:

```text
匿名痕跡は、名前が出ない位置情報です。

誰の痕跡かは分かりません。

1つだけでは弱い情報ですが、複数の痕跡をつなぐと、移動方向や待ち伏せ場所を推理できます。
```

詳細:

```text
主な発生源:

・定期匿名痕跡
・パニック中の痕跡
・監視カメラ通過
・一部の通信混線や傍受
```

## Card: 定期匿名痕跡

タイトル:

```text
定期匿名痕跡
```

一言:

```text
試合中、ときどき誰か1人の位置が名前なしで漏れます。
```

本文:

```text
試合中、全員の中からランダムで1人が選ばれ、その人の近くに匿名痕跡が出ることがあります。

動き続けていると、移動ルートの近くに痕跡が残りやすくなります。

同じ場所に留まり続けると、痕跡が重なって「誰かがここにいる」と読まれやすくなります。
```

詳細:

```text
定期匿名痕跡は、試合時間に応じておおよそ75〜180秒間隔で発生します。

45分試合では、おおよそ108秒に1回が目安です。

対象者はランダムです。

発生するのは名前なしの匿名痕跡です。
```

## Card: 情報の読み方

タイトル:

```text
情報は点ではなく流れで読む
```

一言:

```text
痕跡をつなぐと、相手の動きが見えてきます。
```

本文:

```text
匿名痕跡が1つ出ただけでは、誰の情報か分かりません。

しかし、複数の痕跡、監視カメラ、名前付き暴露を組み合わせると、相手の移動方向が見えてきます。

鬼は痕跡を追い、人側は痕跡を残しすぎないように動くことが大切です。
```

## Detail: よくある誤解

見出し:

```text
よくある誤解
```

本文:

```text
Q. 地図に出た点は現在地ですか？

A. いいえ。多くの場合、現在地ではなく「一度バレた場所」や「痕跡」です。

Q. 匿名痕跡は誰のものか分かりますか？

A. 基本的には分かりません。アナリストなど、一部の役職は読み取り補助がありますが、名前が直接出るわけではありません。

Q. 名前付き暴露は確定情報ですか？

A. その人がその場所で暴露されたことは強い情報です。ただし、今もそこにいるとは限りません。
```

# 7. 鬼との戦い

## Section Header

タイトル:

```text
鬼との戦い
```

一言:

```text
鬼に近づくほど、パニック・拘束・捕獲の危険が高まります。
```

## Diagram

```text
😈 鬼に近いほど危険

🔴 捕獲
🟠 拘束
🟡 パニック
🟢 安全

🧍 逃走者
```

## Card: 危険度の流れ

タイトル:

```text
近づくほど危険になります
```

一言:

```text
パニックは警告、拘束はピンチ、捕獲は脱落です。
```

本文:

```text
鬼に近づくと、まずパニックになりやすくなります。

さらに近づくと接触され、拘束状態になります。

拘束から逃げ切れない、または鬼が至近距離まで近づくと捕獲されます。
```

## Card: パニック

タイトル:

```text
パニック
```

一言:

```text
脱落ではありませんが、匿名痕跡が出て追われやすくなります。
```

本文:

```text
パニックは、鬼の近くにしばらくいると起きる状態です。

パニック中は脱落しません。

ただし、名前なしの痕跡が定期的に残るため、鬼に位置を読まれやすくなります。

「足がすくんで、手がかりを残してしまう」状態だと考えてください。
```

詳細:

```text
パニックは、鬼のパニック圏に約6秒いると発生します。

持続時間は約22秒です。

パニック中は約7秒ごとに匿名痕跡が出ます。

偽位置スキル中は、痕跡がデコイ側に出ることがあります。
```

## Card: 接触・拘束

タイトル:

```text
接触・拘束
```

一言:

```text
鬼に近づかれ続けると、拘束されます。
```

本文:

```text
鬼の接触圏に留まり続けると、接触拘束が始まります。

拘束されても、すぐに脱落するわけではありません。

拘束円の外へ逃げ、一定時間逃げ切れれば助かります。

ただし、逃げ切れない場合は位置が暴露され、捕獲されます。
```

詳細:

```text
拘束円の外に出ても、約10秒以内に戻れば即捕獲にはなりません。

約10秒を超えると、位置暴露後に捕獲されます。

拘束時間はエリア規模に応じて変わります。
```

## Card: 捕獲

タイトル:

```text
捕獲
```

一言:

```text
捕獲されると脱落します。
```

本文:

```text
鬼が至近距離まで近づくと捕獲されます。

GPSでかなり近い場合や、BLE接触がある場合は特に危険です。

捕獲された後も、第二ゲームで試合に関与できます。
```

詳細:

```text
直接捕獲の目安はGPS約12m以内です。

BLE接触がある場合、GPSより強い接近情報として扱われることがあります。

細かい判定は端末や通信状況の影響を受ける場合があります。
```

## Card: 捕獲結界

タイトル:

```text
捕獲結界
```

一言:

```text
地図上に置かれる危険エリアです。
```

本文:

```text
捕獲結界は、スキルで地図上に置く拘束エリアです。

範囲に入ると拘束され、逃げ切れなければ捕獲につながります。

逃走者も鬼も、装備によって使うことがあります。
```

詳細:

```text
捕獲結界の半径は約55mです。

持続時間は約24秒です。

クールダウンは約80秒です。

拘束円の外へ出た場合、約10秒の猶予があります。
```

## Detail: 鬼との距離の目安

見出し:

```text
距離の目安
```

本文:

```text
鬼との危険範囲は、プレイエリアの広さに応じて変わります。

接触圏、拘束円、パニック圏は、広いエリアほど大きくなります。

直接捕獲のGPS距離は約12mが目安です。

正確な範囲は試合設定やエリア規模によって変わるため、画面上の警告を優先してください。
```

# 8. エリア外

## Section Header

タイトル:

```text
エリア外
```

一言:

```text
外に出ても即脱落ではありませんが、長くいるほど危険です。
```

## Diagram

```text
エリア内
──────────── 境界
エリア外

少しの猶予
↓
名前付き暴露
↓
再暴露
↓
脱落
```

## Card: エリア外の基本

タイトル:

```text
エリア外は段階的に危険になります
```

一言:

```text
外に出た瞬間ではなく、外に居続けることが危険です。
```

本文:

```text
プレイエリアの外に出ても、すぐに脱落するわけではありません。

ただし、外に居続けると、名前付き暴露が発生します。

さらに外に居続けると、一定間隔で再び暴露されます。

最終的に、長時間外にいると脱落します。
```

## Card: 名前付き暴露

タイトル:

```text
外にいると名前付きでバレます
```

一言:

```text
エリア外は、逃げ道であると同時に大きなリスクです。
```

本文:

```text
エリア外に長くいると、自分の名前付き暴露が発生します。

つまり、「誰がどこでエリア外にいたか」が相手に伝わります。

エリア端を使って逃げることはできますが、使いすぎると位置を読まれやすくなります。
```

## Card: 鬼のエリア外

タイトル:

```text
鬼もエリア外で脱落します
```

一言:

```text
鬼だけが特別に許されるわけではありません。
```

本文:

```text
本鬼も、エリア外に居続けると脱落します。

ただし、鬼がエリア外に出た瞬間に人陣営の勝ちになるわけではありません。

本鬼が全員いなくなったとき、人陣営の勝ちになります。
```

## Card: 安全地帯チャージ

タイトル:

```text
安全地帯チャージで暴露を防ぐ
```

一言:

```text
チャージがあれば、エリア外暴露を1回防げることがあります。
```

本文:

```text
安全地帯でチャージを得ていると、エリア外による名前付き暴露を1回防げる場合があります。

エリア端を使って逃げるときの保険になります。

ただし、エリア外脱落そのものを無限に防げるわけではありません。
```

## Detail: エリア外の正確な流れ

見出し:

```text
エリア外の詳細
```

本文:

```text
・境界から約25m以内なら、エリア外タイマーは進みにくくなります。
・エリア外が約8秒続くと、警告段階を超えます。
・安全地帯チャージがない場合、名前付き暴露が発生します。
・外に居続けると、約25秒ごとに再暴露されます。
・約90秒続くと脱落します。
・鬼も同じようにエリア外脱落します。
・鬼のエリア外は即人陣営勝利ではなく、鬼の脱落として処理されます。
```

```text
安全地帯
```

## Card: 情報屋

```text
情報屋
```

一言:

```text
相手の手がかりを得られる施設です。
```

```text
情報屋は、相手陣営の情報を得るための施設です。

本鬼は、逃走者1人の位置を名前付きで暴露できます。
```

```text
情報屋の半径は約30mです。

逃走者の使用クールダウンは約120秒です。

本鬼の使用クールダウンは約90秒です。

情報屋の効果は、ホスト設定や情報屋モードによって変わることがあります。
```

```text
監視カメラ
```

```text
監視カメラの近くを通ると、匿名痕跡が残ります。

誰が通ったかは基本的に分かりません。

通る側にはリスクですが、追う側には重要な手がかりになります。
```

詳細:

```text
監視カメラの検知半径は約18mです。

同じカメラの再検知には約90秒のクールダウンがあります。

復讐の鬼影によって停止されることがあります。
```

## Card: 通信妨害ゾーン

タイトル:

```text
通信妨害ゾーン
```

一言:

```text
情報戦をかき乱すエリアです。
```

本文:

